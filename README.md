# PBL2
AI-Driven Framework for Air Quality Prediction and Precautionary Health Recommendations

Random Forest Regression model to calculate a Health Impact Score based on predicted pollution levels and meteorological data.

The objective is to assist policymakers, public health officials, and environmental researchers in making data-driven decisions through accurate forecasting and risk evaluation. The system supports real-time monitoring, provides interpretable visualizations, and is designed for scalable deployment via IoT and cloud infrastructure.

Key Features:

LSTM-based multivariate time-series modeling for AQI prediction

Random Forest regression for quantifying health impact scores

Real-time data ingestion and prediction capability

Interactive data visualization including time-series plots, correlation heatmaps, and feature importance

Cloud-ready architecture with support for IoT-based sensor networks
